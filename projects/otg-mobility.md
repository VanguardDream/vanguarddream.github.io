# Development of Floating Off-The-Ground (OTG) Mobility Technology for New Personal Mobility

**Funded by:** MOTIE  
**Period:** 2020 – 2022  
**Keywords:** OTG mobility, Dynamics, Stability control

---

## Overview

This project developed a novel **Off-The-Ground (OTG) personal mobility platform** — a vehicle concept that operates partially or fully elevated from the ground surface. The work covered dynamics modeling, control system design, and iterative stability experiments for safe operation.

## Key Contributions

- Developed dynamics model for the OTG platform and designed the corresponding control architecture
- Established test scenarios and log analysis pipeline for safe operation validation
- Conducted iterative stabilization and response characterization experiments
