# Development of Narrow Space Navigation Robot Technology for Detection and Rescue

**Funded by:** MOTIE  
**Period:** 2019 – 2021  
**Keywords:** Field robotics, Search & rescue, Narrow space navigation

---

## Overview

This project developed a robotic system capable of navigating **highly confined spaces** inaccessible to humans — such as post-disaster rubble or industrial narrow passages — for the purpose of victim detection and rescue support.

## Key Contributions

- Defined navigation, perception, and mapping requirements for narrow confined environments
- Built experimental testbeds and defined performance metrics for evaluation
- Analyzed failure cases (localization drift, obstacle interaction, communication loss) and implemented targeted improvements
