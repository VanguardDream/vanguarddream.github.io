# Shape-Adaptive Electro-Adhesive Gripper for Grasping Arbitrary Shaped Objects

**Funded by:** MOTIE  
**Period:** 2018 – 2019  
**Keywords:** Electro-adhesion, Variable friction, Adaptive grasping

---

## Overview

This project developed a **shape-adaptive gripper based on Electro-Adhesion (EA)** capable of grasping objects of arbitrary shape. By applying voltage to the EA film, electrostatic attraction generates variable friction force — enabling stable grasping regardless of object surface geometry.

## Key Contributions

- Characterized EA-based grasping principle and modeled friction variation as a function of voltage, surface material, and curvature
- Designed a systematic experiment matrix covering surface material / curvature / load combinations
- Iteratively tuned design and control parameters to improve grasp stability and repeatability
